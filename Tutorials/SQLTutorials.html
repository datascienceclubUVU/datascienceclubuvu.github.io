Overview

SQL (Structured Query Language) is the primary tool used by analysts and developers to search for data stored in a relational database. Many Relational Database Management Systems (RDBMSs) support ANSI-SQL, or the base form of SQL, and many integrate custom functions and procedures within their respective ecosystems. This is a critical language to understand in the every aspect of Data Science. Let's get started with the tutorials listed
