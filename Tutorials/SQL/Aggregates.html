BEGINNER SQL: AGGREGATE FUNCTIONS



Introduction

One of the primary use cases of SQL is Data Analysis. This can be done in a variety of ways, ranging from looking for trends to finding unique distributions. The most basic type of analysis consists of aggregating values based on groups. In this tutorial, we'll look at the most commonly used aggregate functions in SQL.

Description

The most common aggregate functions used in SQL include SUM, COUNT AVG (Average), MIN, and MAX. These are described in more detail below:

SUM

The SUM function is used to add together numerical values. This is used for comparing totals across groups (e.g., sales by state) and identifying anomalies in your data.

Example

   SELECT state, SUM(sales) AS grossSales, year -- "grossSales" is an alias for the aggregate column
   FROM dbo.sales
   GROUP BY state
   ORDER BY grossSales DESC;

Output


COUNT

The COUNT function finds the number of rows included in the calculation. This is used for locating duplicates in your data and removing unnecessary values.

Example

   SELECT state, COUNT(sales) AS numSales, year
   FROM dbo.sales
   GROUP BY state, year
   ORDER BY numSales DESC;

Output


AVG (Average)

The AVG (average) function finds the total of all the numerical values and divides it by the count of the rows included. This is used for comparing trends across groups.

Example

   SELECT state, AVG(sales) AS avgSales, year
   FROM dbo.sales
   GROUP BY state, year
   ORDER BY avgSales DESC;

Output
