BEGINNER PANDAS: Grouping & Aggregates



Introduction

When conducting data analyses, we cometimes want to shift between levels of detail to look at the data from different perspectives. One way Pandas offers this functionality is by providing grouping and aggregate functions. For sake of continuity, we will be looking at the same aggregate functions as displayed in the SQL tutorial.

The "GroupBy" Function

As mentioned many times before, Pandas and SQL have A LOT in common. This is very much true for the GROUP BY clause, simply because Pandas has a function called "groupby". However, unlike SQL, this function has a slightly more complex syntax:

df.groupby([column or list of columns to group by])[numerical columns to aggregate].[aggregate function]()

Example

car_df.groupby(['Make', 'Model'])['Odometer'].sum()

Aggregate Functions

Just like SQL, Pandas offers various functions to aggregate data. The most common aggregate functions include the following:

  • sum()
  • count()
  • mean()
  • min()
  • max()

Because we already learned about the differences and use cases of these aggregate functions, we'll make this tutorial a bit shorter and give you an opportunity to practice them right now.
