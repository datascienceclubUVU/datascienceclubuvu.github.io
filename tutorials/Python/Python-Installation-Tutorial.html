Python Tutorials: Installing Python

Before we start learning Python, we need to setup our environment. For these tutorials, we will be using VS Code on WINDOWS, though you are welcome to use PyCharm, Python's Default IDLE, or another Integrated Development Environment (IDE). This tutorial will cover the following learning objectives:

  • How to Install Python
  • How to Setup Python in VS Code

How to Install Python


DISCLAIMER: There is a sponsored portion of the video below, To avoid this, stop the video at the 3:55 mark.



Summary

  • Before going through the installation process, you need to check if you already have Python installed. to do this, complete the following steps:
    1. Open "Windows Powershell".
    2. Type python --version.
      • If Python is installed, you'll get the following message: python [version]
  • Once you've confirmed Python hasn't been previously installed, complete the following steps:
    1. Go to www.python.org
    2. Hover over the Downloads Menu and Select Download for Windows.
    3. Open your File Explorer and navigate to the Downloads folder.
    4. Click on the Python application you just downloaded to open the Installation Wizard.
    5. Once the Installation Wizard has opened, click the primary Install Now option.

How to Setup Python in VS Code




Summary:

  • If you don't have VS Code currently installed, complete the following steps:
    1. Go to code.visualstudio.org.
    2. Click the Download for Windows button.
    3. Open your File Explorer and navigate to the Downloads folder.
    4. Click on the VS Code application you just downloaded to open the Installation Wizard.
    5. Follow the Wizard's instructions, keeping the default settings to avoid errors.
    6. Open the Application to ensure it installed correctly.
  • Once you successfully have VS Code installed, create a test Python file by completing the following steps:
    1. Go to the File tab and select New File.
    2. Click Select a Language and choose Python.
    3. Press CTRL + "S" to save the file.
    4. Save the file in your File Explorer as a Python file (extension .py).
  • To ensure your Python environment is setup correctly, install the Python and Code Runner VS Code extensions as shown in the video.
