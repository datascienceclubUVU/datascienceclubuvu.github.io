BEGINNER PANDAS: INTRO TO PANDAS



Introduction

Pandas is a Python library that is widely used for data analysis, manipulation, and ingestion. Built on top of the NumPy Python Library. Pandas includes a multitude of methods, classes, and data structures. In this tutorial, we will be taking a look at the different data types and features of Pandas.

Data Types

Just like Python, Pandas includes a variety of data types. Listed below are some of the most commonly used data types:

  • String (str): A string consists of a collection of letters, numbers, and/or symbols contained between two quotation marks.
       Example: 'This is string!'

  • Integer (int): A integer is simply a whole number. In Pandas, integers are classfied as one of the following: int16, int32,
               int64. The disttinction is in the size of the integer. These are used to save memory.
       Examples: 4, 7, 90, 895670

  • Float (float): A float (or floating point number) is a number with n number of decimal places. These are widely used in calculations.
       Examples: 4.25, 7.50, 90.00, 3.1415926

  • Boolean (bool): A boolean value has only two possible outcomes: true or false. These are used for logical tests and conditional statements.
       Examples: true, false

Data Structures

Base Python only provides abstract one and two-dimensional data types. However, Pandas takes those data types and beefs them up to add a Graphical User Interface (GUI) for interaction with the data. Listed below are the most commonly used data structures:

  • List: A list consists of a collection of items with varying data types. These are used for categorical items or small datasets.
       Example: ['First Item', 80, 99.85, false]

  • Array: An array consists of a collection of items with the SAME data types. These are used for static values such as discounts or tax rates.
       Example: ['A', 'B', 'C', 'D', 'E', 'F']

  • Dictionary (dict): A dictionary consists of a collection of key value pairs. Think of this as a dictionary in real life:
                   there's a word (the key) and its definition (the value).
       Example: {'Apple': 'A type of fruit.', 'Carrot': 'A type of vegetable'}
