SQL Tutorials: DML

DISCLAIMER: Before starting this tutorial, it's highly recommended that you download an RDBMS to follow along. We recommend using either PostgreSQL, MS SQL Server, or MySQL.

Congratulations on creating your first database! Now that you have a clear understanding of the first third of SQL, let's continue the journey. Data Manipulation Language (DML) is a form of SQL used specifically for inserting, updating, selecting, and deleting data stored in database objects. Listed below are the learning objectives in this lesson:

  • Inserting Data
  • Updating Data
  • Deleting Data

Inserting Data

Once you have your initial database, schema, and tables, you'll want to start adding data. Watch the following videos to get a better understanding of this concept:

    How to Insert Data into a Table
    How to Insert Data from CSV to SQL Server

Summary:

  • To insert a single row into a database table, use the following syntax (may differ between RDBMSs). It should be noted that the order of the list of values MUST follow the order of the columns when you created the table.
    INSERT INTO [table_name] VALUES ([col_value], [col_value], etc.);
  • To insert multiple rows into a database table, use the following syntax (may differ between RDBMSs):
    INSERT INTO [table_name] VALUES ([col_value], [col_value], etc.), ([col_value], [col_value], etc.);
  • To import CSV data into a database, use the following syntax (differs between RDBMSs):
    BULK INSERT [table_name] FROM '[file_path]'
    WITH (FIELDTERMINATOR='[delimiter]', ROWTERMINATOR='\n',
    FIRSTROW=[first non-header row]);

Updating Data

As your business grows, you may need to make changes to customer or employee data based on industry changes or natural changes such as someone moving addresses. Watch the following video to get a better understanding of this concept:

    How to Update Data in a Table

Summary:

  • To update all the rows in a column, use the following syntax:
    UPDATE [table_name] SET [col_name] = [new_value];
  • To update specific rows in a column, use the following syntax. We will learn more about the WHERE clause in the next tutorial.
    UPDATE [table_name] SET [col_name] = [new_value] WHERE [condition];
  • To update multiple columns in the same statement, use the following syntax:
    UPDATE [table_name] SET [col_name] = [new_value], [col_name] = [new_value];

Deleting Data

Similar to the previous section, as your business grows, you'll naturally lose customers and employees, which may require deleting sensitive data that's no longer necessary. Watch the following video to get a better understanding of this concept:

    How to Delete Data From a Table

Summary:

  • To delete all the data in a table, refer to the TRUNCATE statement in the previous tutorial.
  • To delete specific rows in a table, use the following syntax:
    DELETE FROM [table_name] WHERE [condition];
  • Make sure to double-check your code before executing the statement.
