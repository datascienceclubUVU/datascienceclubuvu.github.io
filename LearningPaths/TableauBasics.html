DATA VISUALIZATION: INTRO TO TABLEAU 2022.1



Introduction

Data visualization has become a fast growing landscape as the field of big data continue to grow. One of the leading technologies in this field is Tableau, a business intelligence and data visualization software that allows users to connect to a wide variety of data sources, create beautiful reporting dashboards, and implement custom actions and parameters for functionality purposes. In this tutorial, we will dicuss the following topics:

  • Installing Tableau
  • Data Source Connections
  • Tableau Objects
  • Common Terminology

Installing Tableau

Complete the following instructions to install Tableau 2022.1 on your local machine:

  1. Go to the Tableau Download Page and click on the Download link.
  2. Once you have the application download, follow the instructions of the Installation Wizard to install the application.
  3. Open the application and fill out the information on the Registration page. This is simply for a two-week free trial that you can renew at any time.

Data Source Connections

As mentioned above, Tableau offers a wide variety of data source connections. For the purposes of this tutorials, we will only be going over the following data source connections:

  • Text Files
  • Excel Workbooks
  • Non-Cloud Relational Databases


Text FIles

Text Files include CSV (comma-separated values), TXT, and TWB (Tableau WorkBooks) files. These are widely used for storing small datasets and samples for statistical analyses. To access a text file from your file explorer, complete the following instructions:

  • On the welcome screen, see screenshot above, select "Text File" from the "Connect" Pane.
  • In the "Select File" dialog box, navigate to and select the file you'd like.
  • Click the "Open" button in the dialog box or double click on the file's icon.

Excel Workbooks

Excel Workbooks are widely used for completing ad-hoc (as needed) data analysis and provide a wide range of data analysis and manipulation capabilities. To access a local excel workbook, complete the following instructions:

  • On the welcome screen, see screenshot above, select "Excel" from the "Connect" Pane.
  • In the "Select File" dialog box, navigate to and select the file you'd like.
  • Click the "Open" button in the dialog box or double click on the file's icon.

Non-Cloud Relational Databases

Relational Databases are widely used to house transactional data and are great for completing basic queries. This is by fra the most common data source connection used for ad-hoc analyses such as the ones we will be completing. To connect to a Non-Cloud Relational Databaxe, complete the following instructions:

  • On the welcome screen, see screenshot above, Under the "Connect" Pane, in the "To a Server" section, select the appropriate database connection.
  • In the "Connect to Database" dialog box, fill out the server, user name, and password fields to create a valid connection.
  • Click the "Connect" button in the dialog box.
