BEGINNER SQL: AGGREGATE FUNCTIONS



Introduction

One of the primary use cases of SQL is Data Analysis. This can be done in a variety of ways, ranging from looking for trends to finding unique distributions. The most basic type of analysis consists of aggregating values based on groups. In this tutorial, we'll look at the most commonly used aggregate functions in SQL.

Description

The most common aggregate functions used in SQL include SUM, AVG (Average), COUNT, MIN, and MAX. These are described in more detail below:

SUM

The SUM function is used to add together numerical values. This is used for comparing totals across groups (e.g., sales by state) and identifying anomalies in your data.

Example

   SELECT state, SUM(sales) grossSales -- "grossSales" is an alias for the aggregate column
   FROM dbo.sales
   WHERE state != 'Puerto Rico' -- The "!=" symbol means "not equal to"
   GROUP BY state
   ORDER BY grossSales DESC;
