BEGINNER PANDAS: Joins & Unions



Introduction

This tutorial assumes the user has previous knowledge of Relational Databases. Just like Relational Databases, Pandas allows you to join and union DataFrames and DataFrame queries to create comprehensive datasets. In this tutorial, we will be going over how to apply different types of joins and how to union multiple DataFrames.

Joins

Just like in SQL, there are multiple types of joins. Rather than going over those types of joins, let's go over the function that makes it all happen: merge(). The merge function has the following syntax:

df.merge([second dataframe], how=[join type], on=[column to join on], left_on=[left column to join on]*, right_on=[right column to join on]*)
* Use these arguments if the columns to join on have different names.

Example

car_df.merge(owner_df, how='inner', left_on='owner_id', right_on='id')

Unions

Pandas and SQL have A LOT in common, however there are also some differences. This is very true for unions. Pandas allows you to union both dataframes and queries. This is done using the concat() function. The concat function has the following syntax:

pd.concat([[dataframe1], [dataframe2]])

Example

pd.concat([car_df, owner_df])
